# Jeu du Juste Prix (Python - Console)

## Description
Ce projet est un jeu en ligne de commande inspiré du "Juste Prix".  
Le programme génère un nombre aléatoire et le joueur doit le deviner.

## Objectif
J'ai réalisé ce projet pour m'entraîner à :
- utiliser des boucles (`while`)
- manipuler des conditions (`if / elif`)
- gérer les entrées utilisateur
- utiliser des nombres aléatoires

## Fonctionnalités
- Génération d'un nombre aléatoire entre 1 et 1000
- Système d'essais
- Indications pour aider le joueur :
  - "Plus" si le nombre est trop petit
  - "Moins" si le nombre est trop grand
- Compteur du nombre de tentatives

## Fonctionnement
1. Le programme choisit un nombre aléatoire
2. Le joueur propose une valeur
3. Le programme indique si c'est plus ou moins
4. Le jeu continue jusqu'à trouver le bon nombre

## Technologies utilisées
- Python
